آغازه یک وب‌اپلیکیشن ساده برای یافتن اولین نسخه آرشیوشده‌ی وبسایت‌ها با استفاده از سرویس Wayback Machine اینترنت آرشیو است.

پیش‌نمایش:

https://aghazeh.maskoot.ir

---

## امکانات

- یافتن اولین نسخه ذخیره‌شده هر وبسایت
- رابط کاربری فارسی و انگلیسی
- طراحی واکنش‌گرا (Responsive)
- اتصال مستقیم به API اینترنت آرشیو
- نمایش پیش‌نمایش نسخه آرشیوشده
- بدون نیاز به پایگاه داده
- اجرا بر بستر Cloudflare Workers
- سبک و سریع

---

## نحوه عملکرد

پس از وارد کردن آدرس یک وبسایت:

1. ورکر درخواست را به CDX API اینترنت آرشیو ارسال می‌کند.
2. قدیمی‌ترین نسخه موجود پیدا می‌شود.
3. تاریخ ذخیره نسخه نمایش داده می‌شود.
4. نسخه آرشیوشده داخل برنامه نمایش داده می‌شود.

## راه‌اندازی

این پروژه برای استقرار دستی در Cloudflare Workers طراحی شده است.

### ایجاد Worker

1. وارد پنل Cloudflare شوید.
2. به بخش زیر بروید:

Workers & Pages → Create Worker

3. محتوای فایل `worker.js` را جایگزین کد پیش‌فرض کنید.
4. روی Save و سپس Deploy کلیک کنید.

---

### اتصال دامنه

از بخش:

Workers & Pages → Triggers

می‌توانید دامنه یا Route دلخواه را به Worker متصل کنید.

مثال:
aghazeh.example.com/*

## محدودیت‌ها

ممکن است برخی وبسایت‌های آرشیوشده به درستی نمایش داده نشوند، زیرا:

- برخی فایل‌های آرشیوشده موجود نیستند.
- محدودیت‌های امنیتی مرورگر مانع بارگذاری می‌شوند.
- برخی صفحات به‌صورت کامل در آرشیو ذخیره نشده‌اند.
